# Homebrew Tap for Prompter CLI

This is the official Homebrew tap for [Prompter CLI](https://github.com/imdevan/prompter), a CLI tool for assembling AI coding prompts.

## Installation

```bash
# Add the tap
brew tap imdevan/prompter

# Install prompter
brew install prompter
```

## Usage

After installation, you can use the `prompter` command:

```bash
prompter --help
prompter --version
```

## Updating

To update to the latest version:

```bash
brew update
brew upgrade prompter
```

## Uninstalling

```bash
brew uninstall prompter
brew untap imdevan/prompter
```
